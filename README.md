# AI Study Assistant 📚🤖

AI-powered study assistant built with Flask that helps students turn lecture notes and PDFs into interactive study material.

## Features

- Upload PDF lecture notes
- Automatic text extraction
- Structured summaries
- Key concept detection
- Quiz question generation
- Flashcard study mode
- Ask questions about uploaded notes
- Interactive learning workflow

## Tech Stack

- Python
- Flask
- HTML
- CSS
- JavaScript
- PyPDF2

ai-study-assistant/

## How to Run

Clone repository:

```bash
git clone https://github.com/DenisBallgjini/ai-study-assistant.git
```

Install requirements:

```bash
pip install flask PyPDF2
```

Run:

```bash
python3 app.py
```

Open:

```text
http://127.0.0.1:5000
```
